Exemplos de posicionamentos

Paragrafo 1 - Posicionamento - Static - não muda, permanece padrão.

Paragrafo 2 - Posicionamento - Relative - O elemento principal e todo movido, mas o espaço anteriormente ocupado continua alocado na memória. não se pode preencher esse espaço com outro elemento

Paragrafo 3 - Posicionamento - Absolute - a posiçao do elemento vai estar relativamente ao elemento pai mais proximo que tenha uma posição diferente de estatica. Será usada quando se quer colocar um elemento num lugar muito especifico que irá ignorar o layout da pagina, como se fosse uma figura por cima de tudo

Paragrafo 4 - Posicionamento - Fixed - O elemento que ele esta posicionado,'o filho' deixa de obedecer as regras do pai para obedecer a janela do navegador (viewport) Eu estive aqui! Paragrafo 4 - Posicionamento - Sticky - vai conbinar, ele tem um comportamento parecido com relative e fixed o elemento vai ser posicionado de forma relativa ate que a pgina seja rolada ate certo ponto dai ela gruda.
